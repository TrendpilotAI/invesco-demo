### Key Technical Learnings
- SLDS utility icons from npm package use 52×52 coordinate system, NOT 520×520
- Next.js `Image` component auto-handles basePath; raw `<img>` does not
- `process.env.NODE_ENV` doesn't work in client components for static export basePath detection
- When extracting icons from SLDS SVG sprite, always check the actual coordinate values in paths to determine correct viewBox
