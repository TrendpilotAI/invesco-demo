**2. Quality Gate v1 — Built, Self-Critiqued, Rebuilt**
First draft: 4 scripts + 7.5KB process doc. Self-review found:
- Pre-commit hooks don't work for agents (sandbox environment)
- review-pr.sh was just grep pretending to be a reviewer
- agent-pr.sh had race conditions, no idempotency
- CI templates had naive assumptions (70% gate on 0% repos)
- Timing budget was fiction

**Rebuilt as single script:** `quality-check.sh` — 7.4KB, no frameworks, no MCP.
- Detects language (Python/TypeScript) including nested project structures
- Secrets scan (API keys, private keys, .env files)
- Lint (ruff for Python, eslint + tsc for TypeScript)
- Tests (pytest with coverage, vitest/npm test)
- Security audit (pip-audit, npm audit)
- Coverage ratchet (baseline on first run, can't go down, slowly goes up)
- Proven on 3 real repos — every failure was a real issue
