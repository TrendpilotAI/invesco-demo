### Key Learnings
- Subagents sometimes corrupt JSX (unmatched divs). Always verify build passes before deploying subagent output.
- Opus subagent failed silently (1-second runtime) — model availability issue. Sonnet 4-6 was reliable fallback.
- `npm install` broken in sandbox env ("Class extends value undefined"). Use unpkg CDN to fetch packages directly.
- GitHub Pages may use "GitHub Actions" source — need to add deploy workflow, not just push to branch.
