## 🔥 If It Breaks — Recovery Plan

| Situation | Do This |
|---|---|
| **Page won't load** | Switch to mobile hotspot. If still broken, say: "Let me show you the recorded walkthrough — same experience" and play Loom recording. |
| **Advisor cards don't render** | Add `?reset=true` to URL and refresh. If still broken, go to Loom. |
| **Mobile view broken** | Skip mobile, describe it verbally: "On mobile it's the same brief, optimized for phone." Move on. |
| **Zoom screen share fails** | Stop share, reshare. If stuck, paste the URL in chat: "Feel free to click through while I reshare." |

**Golden rule:** Don't apologize excessively. Pivot confidently. One sentence max: "Let me pull up the recorded version" — then keep talking.
