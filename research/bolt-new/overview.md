# Bolt.new — Executive Summary

## What Is It
Bolt.new is an AI-powered, browser-based full-stack web application builder by StackBlitz. Users type a natural language prompt, and Bolt generates, runs, and deploys a complete web app — all inside the browser, with zero setup.

## The Story
- **Founded:** StackBlitz started in 2017 (Eric Simons & Albert Pai) as an online IDE
- **The Dark Years:** 7 years building WebContainers tech, stalled at ~$700K ARR, nearly shut down in late 2023

- **Explosive Growth:** $1M ARR week 1 → $4M ARR month 1 → $20M ARR month 3 → $40M ARR month 5
- **Users:** 5M+ users, adding 1M+ per month

## Core Technology: WebContainers
The 7-year moat. WebContainers is a WebAssembly-based micro-OS that runs Node.js entirely in the browser:
- No cloud servers needed per session (huge cost advantage)
- Near-zero latency (feels like localhost)
- Security by default (code runs in browser sandbox)
- Built with Rust → WebAssembly for near-native speed
- Custom file system, process model, networking layer, ESM support

## Why It Worked
1. **Right tech at the right time** — 7 years of WebContainer work met Claude 3.5 Sonnet's coding ability
2. **Zero friction** — No signup, no install, instant "aha moment"
3. **Viral by nature** — People share what they build
4. **Cost structure advantage** — Compute runs on user's browser, not StackBlitz servers
5. **Memorable domain** — bolt.new (Google .new TLD = "create something now")
6. **Generous free tier** — 1M tokens/month, enough to build real things
